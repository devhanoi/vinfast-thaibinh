# VinFast Thái Bình — SEO Landing Page

Landing page tối ưu SEO nhắm mục tiêu **Top 3 Google** cho từ khóa cốt lõi **"vinfast thái bình"** (và cluster liên quan).

## Mục tiêu kinh doanh
- Sinh lead chất lượng cao cho đại lý 3S VinFast tại Thái Bình (mua xe, lái thử, trả góp, bảo dưỡng).
- Xếp hạng Top 3 SERP Google.com.vn trong vòng 60–90 ngày sau khi launch.
- Core Web Vitals "Good" trên cả Mobile và Desktop (LCP ≤ 1.8s, CLS ≤ 0.05, INP ≤ 200ms).

## Đối tượng người dùng
Khách hàng cá nhân/doanh nghiệp tại Thái Bình và các tỉnh lân cận (Nam Định, Hưng Yên, Hải Phòng, Hải Dương) đang tìm hiểu xe điện VinFast: VF3, VF5, VF6, VF7, VF8, VF9, dòng xe dịch vụ (Minio Green, Herio Green, Limo Green), và xe máy điện.
