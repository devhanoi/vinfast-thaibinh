Hãy kiểm tra route được chỉ định và báo cáo theo format này:

| Hạng mục | Kết quả | Ghi chú / Đề xuất fix |
|---|---|---|
| Title (≤60 ký tự, chứa KW) | ✅/⚠️/❌ | … |
| Meta description (140–160, chứa KW + CTA) | ✅/⚠️/❌ | … |
| 1 H1 duy nhất, chứa KW | ✅/⚠️/❌ | … |
| H2/H3 dùng biến thể KW | ✅/⚠️/❌ | … |
| Canonical đúng | ✅/⚠️/❌ | … |
| OG image 1200×630 | ✅/⚠️/❌ | … |
| ≥1 JSON-LD relevant | ✅/⚠️/❌ | liệt kê schema types |
| Internal link ≥3 anchor có KW | ✅/⚠️/❌ | … |
| Hero image `priority` + alt có KW | ✅/⚠️/❌ | … |
| Sitemap entry + lastmod | ✅/⚠️/❌ | … |

Sau cùng đề xuất P0/P1/P2 các fix (P0 = chặn launch).
